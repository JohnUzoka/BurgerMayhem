Troubleshooting (GitHub Pages)

Asset paths and filenames on GitHub Pages are case-sensitive. Ensure images exist under assets/ next to index.html and match exactly:

  • assets/pepe.png
  • assets/hamburger.svg
  • assets/kid.png
  • assets/mcds.jpg

If a link above returns 404, rename the file in the repo to match the exact name and extension, commit to main, and wait a minute for Pages to redeploy.
